# 什么是UV什么是拓扑？

UV=模型展开后的2d空间坐标

拓扑=等于点线面的结构，用于绑定、动画等

# 什么是绑定？什么是权重？

# 材质贴图是什么？

材质=物体的综合表现，比如金属，石头，刺绣

贴图=表现物体质感的素材
