# 什么是UV什么是拓扑？

UV=模型展开后的2d空间坐标

拓扑=等于点线面的结构，用于绑定、动画等

# 什么是绑定？什么是权重？

绑定就是绑骨，权重就是骨头所控制的区域


![image.png](https://cdn.jsdelivr.net/gh/ymingZ/note-gen-image-sync@main/2025-08/022e0305-052a-43ec-b409-31210511a2a7.png)






# 材质贴图是什么？

材质=物体的综合表现，比如金属，石头，刺绣

贴图=表现物体质感的素材

# 什么是动作？

服务于镜头，所以做动作前先决定镜头

# 什么是灯光渲染？

物体只有在合适的灯光下才有表现力
